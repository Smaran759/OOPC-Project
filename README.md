This is a OOPC Mini Project which is a Cashier Interface with Previous Backup log as well as Tax Calculation and Discout Option
To execute this file 
Copy this repo in your Host Machine
Also make sure you have ncurses installed on your machine
Open terminal in the folder and paste this command in it(make sure you have g++ compiler)
g++ ~/OOPC\ Project/ncurses.cpp -lncurses -std=c++17 -o ledger && ./ledger
